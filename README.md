## 2. git LFS 설정

git lfs 설정을 한다면 최대 100MB 까지의 데이터도 협업할 수 있다.
단, 그 이상은 좀 힘들다..

**git lfs 설치하기 바람**

```bash
git lfs pull 
위 명령어를 통해서 clone 받고나서 실행하기 바람
```
